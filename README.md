## Intruções
1. Baixar Stack na Sua Maquina: 'curl -sSL https://get.haskellstack.org/ | sh'
2. Compilar Usando 'stack build' na raiz(UFitCG/) do Projeto
3. Executar o Projeto Usando 'stack exec UFitCG-exe' Tambem na Raiz
